JavaScript 面向对象

js对象与原型

JavaScript中对象是属性名与属性值的集合。对象可以访问原型上的属性和功能，能使JavaScript实现类似于面向对象语言中类（class）和继承的编码风格。

在javascript 中每个对象都有原型的引用，在查询指定的属性时，如果对象本身不存在该属性，则可以到原型上查询，而且对象的原型也可以有原型，这样就形成了原型链。

因为函数是第一对象，所以函数都具有一个原型对象。函数原型都有一个constructor属性都指向函数本身，当函数作为构造函数进行调用时，constructor对象的原型就会被设置为new 新创建的对象的原型。

对象对于函数原型之间的引用关系是发生在对象创建时建立的，如果对象创建后再重写函数的原型对象，之前创建的对象也只会保持原有的原型。并且由于对象与函数原型之间是引用关系，所以当多个对象由一个构造函数创建时，如果修改一个对象的原型属性会影响所有的对象。
